Project Overview

About Visual Sentinel

Visual Sentinel is a graduation research project that addresses the growing challenge of drone detection and 3D localization in real-world environments. The system was designed from the ground up as an integrated hardware-software solution, operating fully at the edge without cloud dependencies.

The project consists of 70 structured experiments producing 140 stereo video pairs, each capturing a static drone at known height and depth coordinates under varying baselines and focal configurations. This dataset forms the backbone of the ML training pipeline.

An embedded orchestration system written in C coordinates dual Raspberry Pi units for synchronized acquisition, IMU sampling, and GPS logging. Detection is handled by a fine-tuned YOLOv8 model, with 3D estimation refined through a residual MLP regression network trained via 5-fold cross-validation.

The evaluation framework uses Leave-One-Out and K-Fold cross-validation to produce honest, leakage-free accuracy metrics. The final system achieves competitive H and D estimation accuracy across diverse geometric configurations.
